What is "Boutime"?

"Boutime" web application allows users to track their working time using a built-in timer.

How it works?

It is easier than it looks! Just go to the "Timer" page, and press "Start" button to start timer.

Also you can Pause and Reset timer whenever you want by clicking matching button

Who made "Boutime"?
